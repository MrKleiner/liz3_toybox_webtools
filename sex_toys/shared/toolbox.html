Python-like range(). Generator function. It's very important to use of and not in

lizard.rndwave

Generate a random string or number
  • Length: 8. Length of the output string
  • Method: 'def'. Choose a predefined character set
    • def: 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789_-'
    • flac: 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789_-()=+*#/!&?<>$~'
    • num: '1234567890'
  • Addchars: ''. Additional characters to use. The input is converted to string and split into characters
  • Crypto: true. True = generate better results with window.crypto at the cost of 50% decrease in performance
lizard.rndwave(20) 'r87a9WosskQ9EFmrHAOZ'

lizard.wait_elem

Wait for a certain selector to appear on the page (useful for writing custom scripts for websites)
First you have to create the control object:
  • Selector: css selector, like '.inner_content #channel_section'
  • Identifier: an id to identify this task in a global pool of similar ones
var load_waiter = lizard.wait_elem('.inner_content #channel_section', 'pg_init_wait')
Returns a control object with the following methods:
  • wait: returns a Promise resolved after the required element appears on the page
  • abort: stop waiting for this element
