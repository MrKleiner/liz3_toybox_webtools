str(65535) '65535'

int()

Python-like int(). Can take anything as an input and tries to evaulate an int
int('65535') 65535 int(3.14) 3

float()

Python-like float(). Can take anything as an input and tries to evaulate a float

len()

Python-like len(). Can take anything as an input and tries to evaulate a length of the object
len('lizards') 7 len({'iguana': 'reptile', 'carp': 'fish'}) 2

range()

Python-like range(). Generator function. It's very important to use of and not in
  • Start: 0.
  • Stop: null (use start as amount)
  • Step: 1. Increase the output number by this amount
for (let idx of range(3)){ console.log(idx) } 0 1 2 for (let idx of range(3, 6)){ console.log(idx) } 3 4 5

lizard.rndwave

Generate a random string or number
  • Length: 8. Length of the output string
  • Method: 'def'. Choose a predefined character set
    • def: 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789_-'
    • flac: 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789_-()=+*#/!&?<>$~'
    • num: '1234567890'
  • Addchars: ''. Additional characters to use. The input is converted to string and split into characters
  • Crypto: true. True = generate better results with window.crypto at the cost of 50% decrease in performance
example of grabbing keys by searching via values in JSON
lizard.rndwave(20) 'r87a9WosskQ9EFmrHAOZ'
